.. _sacct:

sacct
======

``sacct`` is a scheduler command used to display accounting data for all jobs and job steps in 
the SLURM job accounting log or SLURM database.

Documentation
-------------

Documentation is available on the system using the command

.. code-block:: console

    $ man sacct
